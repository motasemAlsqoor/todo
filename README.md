# ToDo list manager

basic web-based todo app manager

## features

1. add todo items

2. show todo items in a list

3. app setting to control the list

## technology

client : React.js,bootsrap
server : node.js
database : mongodb

## start

npm start

## build

npm build

## test

npm test
